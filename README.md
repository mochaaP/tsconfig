# tsconfig

Shareable TypeScript compiler configuration file (`tsconfig.json`) for mochaaP's projects.

## Usage

Install it:

```
npm i -D @mochaa/tsconfig
```

or

```
yarn add --dev @mochaa/tsconfig
```

Then, edit the `tsconfig.json`:

```json
{
  "extends": "@mochaa/tsconfig"
}
```
